## Whac-A-Mole
## Memory Game

Simple game, made with HTML, CSS and Vanilla JavaScript.

### Learn:
- `addEventListener()`
- `querySelector()`
- `setInterval()`
- `classList`
- `forEach()`
- `Math.random()`
- `Arrow functions`

### How to Play
- Try to whac the mole as many times as you can within 30 seconds
